# AI Email Automation using n8n and Groq

## Overview

This project automates email generation using AI. Users submit a form, an AI agent processes the request, and an email is automatically sent.

## Features

✅ Form submission trigger  
✅ AI-powered email generation  
✅ Structured output parsing  
✅ Automatic email delivery  
✅ No-code AI automation

## Workflow

1. User submits a form.
2. The AI Agent processes the input.
3. The Structured Output Parser formats the response.
4. Gmail automatically sends the email.

## Tech Stack

- n8n
- Groq API
- Gmail
- AI Agent Node
- Structured Output Parser

## Use Cases

- Customer support replies
- Automated inquiries
- Lead follow-up emails
- AI email assistants

## Installation

1. Import the workflow JSON into n8n.
2. Add your Groq API credentials.
3. Connect your Gmail account.
4. Activate the workflow.
